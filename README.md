# Hodgkin Huxley Neuron Model Simulation

This script simulates the membrane potential of an artificial neuron using the Hodgkin-Huxley model differential equations. The model accounts for the dynamics of sodium (Na+) and potassium (K+) ionic channels, as well as the leakage current. An external stimulus current is applied within a specific time interval to observe the generation of action potentials. The simulation results provide a membrane potential vs. time plot, which is useful for analyzing the dynamic behavior of neurons.
